The Challenge

Apex Bank faced a significant hurdle: their existing mobile application was built on legacy infrastructure that resulted in 5-second load times and a convoluted navigation structure. User churn was at an all-time high, and customer support tickets related to mobile login failures were overwhelming the support team. They needed a complete overhaul to compete with digital-first neobanks.

2

Our Solution

We approached this problem by implementing a microservices architecture to decouple the frontend from the heavy legacy backend. This allowed for lighter API calls and a snappier interface. We also introduced a new design system focused on accessibility and "thumb-friendly" navigation.
